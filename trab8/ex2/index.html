Nos dois primeiros exemplos a principal diferença fica por conta do tipo de objeto que o javascript espera na resposta, sendo que no ex1 a resposta é "parseada" e no ex2 o código já espera um objeto do tipo JSON. Em ambos a conexão é feita por método GET.

No terceiro exemplo, o servidor recebe um objeto JSON através de método POST e retorna outro JSON contendo os dados solicitados.
